Make sure you have node installed on your computer

Run frontend
1. Go to [frontend Folder](/frontend/) folder.
2. Make .env file from [.env.example](/frontend/.env.example).
3. Run `npm install` in terminal.
4. Run `npm run start` or `npm start`

Run backend
1. Go to [backend Folder](/backend/) folder.
2. 2. Make .env file from [.env.example](/backend/.env.example).
3. Run `npm install` in terminal.
4. Run `npm run dev` in terminal.
